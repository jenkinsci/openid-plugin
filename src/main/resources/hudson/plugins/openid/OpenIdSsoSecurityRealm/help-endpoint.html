Specify the OpenID provider that is used for authentication.

This URL can be either OpenID provider endpoint URL, or a representative OpenID identifier, in which case the endpoint URL will be discovered from this representative identifier. In other words, if you don't know what the endpoint URL is, just provide a valid OpenID in your provider and let Hudson figure out the rest.

For example, specify https://www.google.com/accounts/o8/id for Google and https://me.yahoo.com/ for Yahoo.
